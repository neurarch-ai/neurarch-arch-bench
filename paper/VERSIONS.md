# Paper versions

- `paper/` — arXiv / preprint (article class, single column). Canonical source.
- `paper-iclr2026/` — ICLR 2026 official template.
- `paper-neurips/` — NeurIPS 2024 official template.

Edit `paper/paper.tex` as the source of truth; the venue versions differ only in
the preamble (document class + title/author macros). REVIEWER_SELF_ASSESSMENT.md
is a candid pre-submission review.
